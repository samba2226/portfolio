
**Interactive Portfolio Website**

This is a responsive and interactive personal portfolio website built using HTML, CSS, and JavaScript.
The project showcases personal information, skills, and contact details while demonstrating core front-end development concepts and JavaScript interactivity.

**Project Overview**

The main goal of this project is to create a clean, user-friendly portfolio website with interactive features.
It highlights practical implementation of:

  Semantic HTML structure
  Responsive CSS design
  JavaScript-based interactivity and form validation

This project is suitable for beginners and demonstrates real-world web development practices.

**Technologies Used**

  HTML5 – Structure and content
  CSS3 – Styling, layout, and responsiveness
  JavaScript (ES6) – Interactivity and logic
  GitHub Pages – Deployment

**Features Implemented**
**JavaScript Features**

  DOM manipulation
  Event handling (click, submit)
  Reusable functions
  Local Storage usage

**Interactive Elements**

  Dark / Light Mode toggle (saved using Local Storage)
  Interactive image slider
  Show / Hide content sections
  Real-time form validation feedback

**Form Validation**

  Email format validation
  Required field checks
  Minimum message length validation
  Dynamic error and success messages
